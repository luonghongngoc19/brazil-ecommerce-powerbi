# Brazilian E-Commerce Sales Dashboard | Power BI

## Overview

This project analyzes Brazilian e-commerce sales performance using Power BI to provide actionable business insights.

## Tools

- Power BI
- Power Query
- DAX

## Key KPIs

- Revenue: 13.59M
- Orders: 99,441
- AOV: 136.68
- Avg Review Score: 4.09

## Key Insights

- Revenue shows strong seasonal trends
- Logistics is the main bottleneck affecting customer satisfaction
- Credit card is dominant payment method
- Revenue is concentrated in top product categories
